# AI 자동 번역 가이드

## 📖 개요

Google Sheets에 중국어 FAQ가 없을 때 GPT-4o-mini를 사용하여 자동으로 번역하는 시스템입니다.

## ✨ 주요 기능

### 1. 자동 번역
- 중국어로 FAQ 조회 시, 시트에 중국어가 없으면 자동으로 AI 번역
- GPT-4o-mini 모델 사용 (빠르고 저렴)
- 대학 행정 용어에 최적화된 번역 프롬프트

### 2. 번역 캐시
- 한 번 번역한 FAQ는 메모리 캐시에 저장
- 동일 FAQ 재조회 시 즉시 응답 (API 호출 없음)
- 비용 절감 및 응답 속도 향상

### 3. Fallback 처리
- AI 번역 실패 시 한국어로 자동 fallback
- 에러가 발생해도 서비스 중단 없음

## 🔧 기술 구현

### 코드 위치
- **번역 서비스**: `app/services/translator.py`
- **API 통합**: `app/routers/faqs.py`
- **캐시**: `_translation_cache` (메모리 딕셔너리)

### 번역 프롬프트
```python
[규칙]
1. 대학 행정 용어는 중국 대학 기준으로 번역
   예) 수강신청 → 选课, 학점 → 学分, 장학금 → 奖学金
2. 고유명사(계명대학교, 장춘대학 등)는 원문 유지
3. 한국 특유의 제도는 괄호 안에 설명 추가
4. 자연스러운 중국어 표현 사용
5. 존댓말·경어체는 정중한 중국어 표현으로 전환
```

|------|---------|------|

| 월 사용량 (조회 1000회, 캐시 활용) | ~15,000 tokens | $0.01~0.02 (₩15~30) |

## 🌐 API 사용법

### 중국어 FAQ 조회
```bash
# 전체 중국어 FAQ
curl "http://localhost:8002/api/v1/faqs?lang=zh"

# 특정 카테고리
curl "http://localhost:8002/api/v1/faqs?category_major=입학/졸업&category_minor=입학원서&lang=zh"

# 개별 FAQ 상세
curl "http://localhost:8002/api/v1/faqs/test-001?lang=zh"
```

### 응답 예시
```json
{
  "items": [
```

## 📈 하이브리드 전략 (권장)

### 1단계: AI 자동 번역 (현재)
- ✅ 즉시 모든 FAQ를 중국어로 제공
- ✅ 유지보수 편리 (한국어만 관리)
- ✅ 일관성 있는 번역

### 2단계: 점진적 수동 번역 (향후)
1. **조회수 상위 FAQ 우선 개선**
   ```sql
   SELECT * FROM FAQ_Master ORDER BY 조회수 DESC LIMIT 20
   ```
   
2. **AI 번역 결과를 검토 후 Google Sheets에 입력**
   - AI 번역을 베이스로 전문가 검토
   - 시트에 입력하면 이후부터는 AI 번역 대신 시트 값 사용
   
3. **전문 용어가 많은 FAQ 우선 수동 번역**
   - 비자/체류, 보험, 학사 관련 등

### 3단계: 완전 수동 번역 (선택사항)
- 모든 FAQ를 수동으로 번역 완료
- AI 번역은 백업용으로만 사용

## 🛠️ 관리 방법

### 번역 캐시 초기화
서버 재시작 시 자동으로 캐시가 초기화됩니다.

### 번역 품질 개선
1. **프롬프트 수정**: `app/services/translator.py`의 `_SINGLE_PROMPT` 수정
2. **용어집 추가**: 프롬프트에 용어 사전 추가
3. **수동 번역**: Google Sheets에 직접 입력

### Google Sheets에 번역 입력
1. Google Sheets에서 FAQ_Master 시트 열기
2. '질문(중국어)', '답변(중국어)' 컬럼에 번역 입력
3. 저장 후 즉시 반영 (캐시 초기화는 서버 재시작 필요)

## ⚠️ 주의사항

### 캐시 제한
- 메모리 캐시는 서버 재시작 시 초기화
- 영구 캐시가 필요하면 Redis 등 도입 고려

### API 의존성
- OpenAI API 장애 시 중국어 서비스 영향
- Fallback으로 한국어 반환

### 비용 관리
- 조회수가 많아지면 캐시 효율이 중요
- 필요 시 Redis 등 외부 캐시 도입
