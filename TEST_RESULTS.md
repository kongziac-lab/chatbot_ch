## 🚦 문제 해결 가이드

### 문제 1: FAQ 버튼이 여전히 안 보임
**해결책 1**: 브라우저 강제 새로고침
```
Mac: Cmd + Shift + R
Windows: Ctrl + Shift + R
```

**해결책 2**: 브라우저 캐시 완전 삭제
1. F12 (개발자 도구)
2. Application → Storage → Clear storage
3. 페이지 새로고침

**해결책 3**: 시크릿 모드로 테스트
```
Chrome: Cmd/Ctrl + Shift + N
```

### 문제 2: "정보를 준비 중입니다" 메시지
**원인**: 오래된 JavaScript 코드
**해결**: 위 해결책 1-3 중 하나 시도

### 문제 3: API 호출 오류
**확인 사항**:
1. FastAPI 서버 실행 중인지 확인
   ```bash
   curl http://localhost:8002/health
   ```
2. 브라우저 콘솔 (F12) → Console 탭에서 에러 확인
